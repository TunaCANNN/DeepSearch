# DeepSearch Usage Guide

## Getting Started

1. Download/clone the repository
2. Open `main.lua`
3. Change `version` to either `"full"` or `"lite"`
4. Execute the script in your executor

## Controls

- **RightShift** → Toggle UI visibility (useful for undetectability)
- Sidebar buttons → Quick access to scans
- Command Bar → Type commands like `scan`, `deep`, `full`, `perf`, `clear`, `exit`

## Scanning Modes

- **Quick Scan** → Fast general scan
- **Deep Scan** → More thorough scan
- **Full Scan** → Scans everything (can be slow)

## Wordbanks

- `main.json` → General keywords
- `priority.json` → High value items (recommended for finding exploits)

## Tips

- Use Performance Mode on weaker executors
- Always check the Logs tab for high-value findings
- Update wordbanks regularly for better results
